用mermaid画类图

key description
<|– 继承
*– 组合
o– 集合
–> 联合
– 链接(solid)
..> 属于
..|> 实现
.. 链接(dashed)
+ 公有属性
- 私有属性
# 受保护属性
~ 封装/内部属性
$ static
* abstract
<<Interface>> 接口类
<<Abstract>> 抽象类
<<Service>> 服务类
<<Enumeration>> 枚举

classDiagram
    class A{
        +int a
        +geta() int
        ~int b
    }
    class B{
        -list~int~ a
        +int b $
    }
    A <|-- B : 类B继承类A
    A *-- C
    A o-- D
